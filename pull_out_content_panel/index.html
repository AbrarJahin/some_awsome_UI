Pull Out Content Panel

IMPORTANT this is a limited preview, you only see a portion of the possibilities of this panel. If you need to see the other options included, be sure to check the description, the video preview and the screenshots.

The panel is initialized with the following script (not the demo) :

    $('#pocp1').pullOutContentPanel({
        pocp_scrollbars : true,
        pocp_showonload: false,
        pocp_pg_overlay: false
    });

This Pull Out Content Panel has been created to offer a complete solution to add side menus on your site. This element can be placed on the left and on the right and it has been carefully designed to fit into any website. The jQuery script offers a few options that you can easily set up : you can enable or disable the custom scrollbars, make the panel visible when the page loads or hide it if a user clicks anywhere else on the page.

If you have a lot of content to add into this panel, you have two options : first you'll see custom scrollbars that will automatically take into account the height of your content. If that's not enough, you can add several panels that can be scrolled with the arrows.

This element includes a lot of styled elements such as headings, lists, toggles, form elements, etc. The font-awesome icon collection is already there to enhance your list elements. The CSS file is well structured and easy to customize, you can change the colors, the panel width and any element that you can see. Images and videos will automatically take the available space so you don't need to resize them manually.

The contact form included is quite simple and you only need to add your email address. The feedback messages appearing when a user sends an email can be changed or translated easily.

Compatibility

This element has been tested on several devices and browsers to ensure a maximum compatibility : Internet Explorer, Firefox, Chrome, Opera, Safari have successfully passed the tests. Here's a list of compatibility :

  • Internet Explorer 7+
  • Firefox 4+
  • Chrome
  • Opera 10.50+
  • Safari 4+
  • Chrome on Android
  • Safari on iPad and iPhone

The whole panel is styled via CSS so you may notice some slight differences between browsers : some CSS3 effects like shadows or rounded corners are not showing up in Internet Explorer for example (for more informations on this topic, you can visit findmebyIP).

Mega Menu Reloaded - Fixed

This Mega Menu comes with awesome options and is the most complete solution to build mega menus. It also works without javascript and when disabled most of the content remains accessible. Colors are easy to change and you have 2 awesome tools that can help you : Colour Lovers to find colors and 0to255 for different tones of a given color.

The menu can work on mouse click or mouse hover and comes with a complete set of options detailed in the documentation.

The contact form doesn't require any PHP knowledge, you only have to add your email address as specified in the documentation. All feedback messages appearing when sending an email can be translated in the source code and are easy to modify. This contact form is quite simple and can be replaced by another product that has more options.

This is a demonstration page and upon purchase you'll get HTML pages with clean indented markup. Integrating and customizing this menu requires at least a basic knowledge of HTML and CSS, this is not a plugin ready to be uploaded to your Wordpress or Joomla site. All modifications have to be done in the source code.

Features

  • Menu Bar Carousel
  • Custom Scrollbars (for large drop downs with a lot of content)
  • Inner Tabs to include more content
  • Column Grid System (up to 12 columns)
  • Relies mainly on CSS, improved with some javascript
  • Clean and attractive design
  • 100% valid HTML/CSS markup
  • Uses some CSS3 effects
  • Comes with a fully working AJAX contact form
  • Cross-browser tested

It has been tested with all the major browsers and degrades gracefully down to IE6. All tests have been done with native applications running under a virtual machine to get accurate results. Please keep in mind that some CSS3 features don't work with old browsers like rounded corners. For more informations, please visit findmebyIP.

This menu is fully responsive and has been tested on an iPhone, an iPad and two other Android devices.

Compatible Browsers

  • Internet Explorer 6+
  • Firefox 2+
  • Chrome 1+
  • Safari 3+
  • Opera 11+

Notice : IE6 requires an additional script that improves its compatibility with some basic CSS properties. You'll find more informations here.

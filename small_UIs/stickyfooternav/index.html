    StickyFooterNav, the most versatile footer !

    StickyFooterNav is a revolutionnary navigation system designed to stick at the bottom of the browser. It's not only a navigation bar but a complete system that supports mega panels (called drop downs in "regular" mega menus).

    It is completely responsive and adapts the layout to the viewing environment by using fluid, proportion-based grids, flexible images, and CSS3 media queries. Here are a few features included :

    • Unique 100% fluid mega footer
    • Responsive system that fits on any device
    • jQuery plugin with options + Custom scrollbars
    • Works on click (recommended) or on hover
    • Tested on Android & iOS tablets & mobiles

    For desktop browsers, tablets and mobiles

    On tablets and mobiles fixed elements are particularly difficult to handle. For this reason StickyFooterNav has been designed and coded to adapt itself according to the environment. Each part of the footer will be scaled properly in order to keep all the content accessible.

Responsive CSS3 Mega Drop Down Menu

This Responsive Mega Drop Down Menu is a flexible and easy to integrate solution to build your custom menus. The drop down relies only on CSS/HTML and comes with a fully working contact form.

3 main variants are included : horizontal, vertical aligned on the left and vertical aligned on the right.

The content can be organized into 6 columns based on the 960 grid system. Since the version 6.0, the column grid works with percentages and is much more flexible. This menu comes with 9 color variants, 2 color schemes for the drop downs – dark and light) and a detailed documentation to help you to integrate it.

IMPORTANT : this demo shows a single variant of the menu, for more example, see the screenshots.

Features

  • 100% Responsive mega menu
  • Working contact form (requires JS files)
  • 9 color schemes
  • Horizontal and vertical layouts
  • Full width drop down
  • Easy to customize
  • Valid HTML/CSS3 markup
  • Cross-browser support
  • Up to 6 columns for your content
  • Styled typography and elements
  • Styled form elements
  • Left or right alignment
  • Help documentation

Compatible Browsers

This menu has been tested (and works !) with all the following browsers :

  • Internet Explorer 7+
  • Firefox 2+
  • Safari 3+
  • Opera 10+
  • Chrome 1+

Mobile Devices

This mega menu is fully responsive since the version 6.0. It has been tested on iPad, iPhone and Android. It's possible to turn off the responsiveness as explained in the documentation.
